# 3D Aim Trainer

一款基于 WebUI 的 3D 练枪软件，使用 **Three.js + HTML + CSS + JavaScript** 实现。  
项目模拟标准 AimLab 六目标训练模式，支持 Valorant / 无畏契约风格灵敏度、准星自定义、目标大小调整、计分与连击系统。

---

## 功能特性

### 1. 3D 练枪场景

- 第一人称视角
- 鼠标锁定 Pointer Lock 控制视角
- 白色大型平面墙
- 黑色球形目标
- 六个目标同时存在
- 目标位于同一垂直平面内
- 命中目标后，该目标消失并随机刷新
- 目标刷新区域限制在墙面中心区域，减少过大幅度拉枪

---

### 2. Valorant 风格灵敏度

灵敏度算法参考 Valorant / 无畏契约的常见换算方式：

```js
角度增量 = 鼠标移动量 × sensitivity × 0.07°
```
